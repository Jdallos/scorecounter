Score Counter App

This is a simple Javascript application designed to track users scores.

This application has some additional functionality:
    -Win by two points, tie break feature.
    -Custom winning point.
    -Padel mode- a score tracker for the popular new sport with updated UI.
    -Custom player names.
    -Integrated timer with switch (automatically reset and start) functionality.
    -Keyboard functionality- left arrow +1 Player One, right arrow +1 Player Two, shift to reset.
    -Fireworks on victory!

Improvements and updates:
